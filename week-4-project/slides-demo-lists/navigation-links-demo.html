Block elements such as the unordered-list tags can have width, height, padding and margin. Inline elements don't apply margin to top or bottom or height unless display value is set to block or inline-block

Since LI (List Item) element is a block element, it will take an entire row. Now look at the emphasis tag below, even though we defined the css properties to give the element some height, it will not apply.

This is an inline element which has a height property, but does not function as desired

The inline element below, emphasis tag now has the same height property, but now we set it's display to inline-block, so we can now control it's dimensions

This inline element now sets its css properties to display it as an inline-block so now the dimensions for top and bottom is now functioning as desired
